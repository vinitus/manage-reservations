## 예약 페이지 프로젝트

### 개발환경

1. windows
2. 런타임 환경 - node 18.18.0
3. 사용 언어 - javascript, typescript
4. 패키지 매니저 - npm
5. 빌드 도구 - vite
6. 프론트엔드 프레임워크 - sveltekit
7. 전역 상태 관리 - svelte-store

기타
prettier, eslint, git

## 개발서버로 시작하는 방법

```bash
npm install

npm run dev -- --open
```

## 빌드 후 프로덕션 버젼으로 시작하는 방법

```bash
npm install

npm run build

npm run preview
```
